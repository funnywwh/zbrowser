# 字体文件说明

本目录包含支持多语言的字体文件：

## 中文（Chinese）
- wqy-microhei.ttc - 文泉驿微米黑（支持中文）
- wqy-zenhei.ttc - 文泉驿正黑（支持中文）
- SourceHanSansSC-Regular.otf - 思源黑体简体中文（支持中文、日文、韩文）
- SourceHanSansSC-Medium.otf - 思源黑体简体中文（支持中文、日文、韩文）

## 日文（Japanese）
- SourceHanSansSC-Regular.otf - 思源黑体（支持中文、日文、韩文）
- SourceHanSansSC-Medium.otf - 思源黑体（支持中文、日文、韩文）

## 韩文（Korean）
- SourceHanSansSC-Regular.otf - 思源黑体（支持中文、日文、韩文）
- SourceHanSansSC-Medium.otf - 思源黑体（支持中文、日文、韩文）

## 泰文（Thai）
- NotoSansThai-Regular.ttf - Noto Sans 泰文
- NotoSansThai-Bold.ttf - Noto Sans 泰文（粗体）

## 阿拉伯文（Arabic）
- NotoSansArabic-Regular.ttf - Noto Sans 阿拉伯文
- NotoSansArabic-Bold.ttf - Noto Sans 阿拉伯文（粗体）

注意：
- Source Han Sans SC 是 OpenType (OTF) 格式，但功能上与 TTF 相同
- wqy 字体是 TTC (TrueType Collection) 格式
- 这些字体都是 Chrome 浏览器常用的多语言字体
